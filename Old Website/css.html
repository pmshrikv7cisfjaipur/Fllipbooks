Overview

It stands for cascading style sheet.it is used to create dynamically changing website.in this webpages may include animations,dynamic menus & text effects. With CSS, you’re going to completely control the presentation of your pages, often without even changing your XHTML.

Like XHTML, you can format your CSS pretty much as you like. For longer rules you’ll usually want to add some linebreaks and indenting to make the CSS more readable (for you).You can have as many rules as you want for an element. Each rule adds to the style information of the rule before it.

*Features of CSS:

1.first impression:initial page grab attention
2.interface design:menu interface interactive and visually interesting.
3.corporate mildew:site have a corporate look for marketing purpose.
4.coriolis effect:site generates enough currents of interest based on design and content for the user to comeback.
5.dynamic content which allows user to dynamically change webpage content.
6.dynamic positioning of webpage element.
7.dynamic style which allows the user to change the webpages color,font,size or content.
8.offers highly personalized and customized visitor options.
9.scripts can read in data sources and display it differntly depending on how it is run.
10.create the illusion of being updated regularly using time and date sensitive routines (or even randomisers) to display pre-written text.
