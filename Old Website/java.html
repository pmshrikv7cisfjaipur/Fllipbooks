Overview

java was developed by james gosling at sun microsystem in 1991. java is a breed of coffee thats why we can see coffee mug in its logo. sun microsystem was founded by 4 people in 1982. vinod khosla was the co-founder of sun microsystem.

java was developed as a solution to a problem. sun microsystem makes sofware for appliances, but their problem was to make a platform independent language, so they developed java. on january 27,2010,sun was acquired by oracle so currently java is product of oracle. slogan of java is "java is everywhere". first commercial version of java came in 1996.

*Features of java:

1.it resides in mobiles, client machines, server machines,embedded device,smart phones, cloud,etc.
2.it shares the same basic features of the language and libraries.
3.principle of java:write once,run anywhere(WORA) which makes it portable.
4.it is a simple language:-automatic memory release.
5.it is 100% object oriented language.
6.it is distributed language:software independent.
7.it is interpreted language:dont need compiler
8.it is a robust language:which means its program has capability to handle any type of situations like exception handling.
9.java is a secure language:money transfer task, data encryption
10.java is portable
